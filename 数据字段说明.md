字段解释

**1) 顶层键（payload 根对象）**

| 键名     | 含义                   | 格式/取值            | 前端用途                                        |
| :------- | :--------------------- | :------------------- | :---------------------------------------------- |
| source   | 数据来源标识           | "mock" 或 "official" | 前端页面当前不直接展示；后端可按来源筛选数据    |
| month    | 榜单所属月份           | YYYY-MM，如 2026-03  | 直接影响“本期/历史月份”榜单内容与右上角月份显示 |
| revision | 同月同来源的数据版本号 | 整数，>=1，可选      | 前端不直接展示；用于后端版本管理/覆盖           |
| items    | 本批次评测条目数组     | 非空数组             | 榜单、首页焦点榜单、详情页的数据主体            |

**2) items[] 每个条目的字段说明**

| 键名                | 含义             | 格式/取值                                                  | 前端页面用途                                   |
| :------------------ | :--------------- | :--------------------------------------------------------- | :--------------------------------------------- |
| id                  | 产品唯一标识     | 非空字符串                                                 | 列表渲染 key、Logo 组件标识、竞技场匹配产品    |
| slug                | 产品详情路由标识 | 非空字符串，建议 URL-safe                                  | 详情页 URL：/evaluations/{slug}                |
| name                | 产品名称         | 非空字符串                                                 | 首页焦点榜单、榜单行、详情页标题               |
| organization        | 机构名称         | 非空字符串                                                 | 榜单展示机构名；Logo 映射也依赖该值            |
| category            | 产品大类         | "金融大模型" \| "金融智能体" \| "MCP" \| "Skill" \| "其他" | 榜单分类切换、筛选查询                         |
| subCategory         | 产品子类         | 非空字符串                                                 | 当前主要用于卡片型组件标签（未来扩展位）       |
| summary             | 产品简介         | 非空字符串                                                 | 详情页简介；卡片模式下也会展示                 |
| status              | 评测状态         | "已评测" \| "评测中" \| "即将发布"                         | 榜单状态 pill 颜色、详情页状态标签             |
| overallScore        | 综合分           | 0~100 整数                                                 | 榜单排序主分、首页 Top 榜、详情页总分          |
| businessScore       | 业务分           | 0~100 整数                                                 | 榜单“业务能力”、详情页业务分                   |
| complianceScore     | 合规分           | 0~100 整数                                                 | 榜单“合规安全”、详情页合规分                   |
| awardTags           | 奖项标签数组     | string[]，可空数组                                         | 详情页奖项 Tag；卡片模式也展示                 |
| certificationLevel  | 认证等级文本     | 字符串，默认“通过评测”                                     | 详情页标签（认证等级）                         |
| scenarios           | 适用场景         | string[]，可空数组                                         | 详情页“适用场景”Tag 列表                       |
| highlights          | 核心亮点         | string[]，可空数组                                         | 详情页“核心亮点”列表                           |
| risks               | 风险提醒         | string[]，可空数组                                         | 详情页“风险提醒”列表                           |
| reportUpdatedAt     | 报告更新时间     | YYYY-MM-DD                                                 | 详情页“报告更新时间”；也可用于“按更新时间排序” |
| scoreBreakdown      | 分项评分明细     | 数组，可空                                                 | 详情页“业务/合规分项得分明细”                  |
| personaFits         | 画像适配结论     | 数组，可空                                                 | 详情页“用户画像适配结果”                       |
| productLogoUrl      | 产品 Logo URL    | 合法 URL，可选                                             | 当前前端不直接使用此字段渲染                   |
| organizationLogoUrl | 机构 Logo URL    | 合法 URL，可选                                             | 当前前端不直接使用此字段渲染                   |

**3) 嵌套对象字段**

scoreBreakdown[] 每项建议结构（虽然你最小 JSON 可先给 []）：

| 键名          | 含义         | 前端用途                                         |
| :------------ | :----------- | :----------------------------------------------- |
| dimensionCode | 维度编码     | 作为明细项 key（技术主键）                       |
| dimensionName | 维度名称     | 详情页明细标题                                   |
| score         | 维度分数     | 详情页分项进度条                                 |
| weight        | 维度权重     | 详情页分项显示“权重”                             |
| comment       | 维度评语     | 详情页维度解释文字                               |
| evidenceType  | 证据类型数组 | 当前前端未展示，建议保留                         |
| group         | 分组         | "business" 或 "compliance"，决定出现在哪个明细栏 |

personaFits[] 每项结构：

| 键名       | 含义         | 前端用途                                 |
| :--------- | :----------- | :--------------------------------------- |
| code       | 画像编码     | 详情页卡片标题                           |
| label      | 画像标签说明 | 详情页展示                               |
| assessment | 适配结论     | 详情页标签色（适配较好/需要谨慎/不建议） |
| note       | 备注说明     | 详情页正文                               |
