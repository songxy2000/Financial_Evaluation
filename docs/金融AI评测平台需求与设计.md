## 四、评测体系

### 通用评测

#### 业务能力评分项

| 评分项     | 说明                         | 权重 |
| ---------- | ---------------------------- | ---- |
| 场景适配度 | 产品与目标金融场景的匹配程度 | 25%  |
| 业务准确率 | 解决实际业务问题的准确程度   | 25%  |
| 效率提升   | 为业务带来的效率改进         | 20%  |
| 集成便利性 | 与金融机构现有系统的对接难度 | 15%  |
| 可扩展性   | 能否快速复制到其他场景       | 15%  |

#### 规与风险评分项

| 评分项       | 说明                   | 权重 |
| ------------ | ---------------------- | ---- |
| 监管合规     | 是否符合金融监管要求   | 25%  |
| 数据安全     | 数据加密、隐私保护措施 | 25%  |
| 模型可解释性 | 决策透明度、可审计性   | 20%  |
| 风险控制     | 风险识别和防范机制     | 15%  |
| 合规文档     | 资质证书、审计报告等   | 15%  |

### 用户画像系统

每道测试题结合用户画像，确保AI产品给出符合适当性要求的答案。

#### 核心维度

| 维度     | 分级                                        | 标记      |
| -------- | ------------------------------------------- | --------- |
| 风险等级 | 保守型 / 平衡型 / 激进型                    | C / B / A |
| 投资经验 | 初级 / 中级 / 高级                          | J / M / S |
| 资产规模 | 小额(<50万) / 中等(50-500万) / 大额(>500万) | S / M / L |
